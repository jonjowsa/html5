Sobre a tag div

Tag usada para definir uma divisão ou seção do documento.

Esta tag geralmente é usada como contêiner para outros elementos.

Muitas vezes o principal objetivo de uma tag div é estilizar com CSS ou executar tarefas com JavaScript.

Acesso a todos os atributos globais.
