Sobre a Tag embed

Tag usada para definir um conteúdo externo embutido.

Herda todos os atributo globais do html5.

Atributos
height Define a altura do conteúdo embutido. Número de pixels
src Define a fonte do conteúdo embutido. URL
type Define o tipo do conteúdo embutido. Tipo da mídia
width Define a largura do conteúdo embutido. Número de pixels
