Sobre a Tag META


Tag usada para definir um metadado da página.
Os metadados são usados pelos navegadores, mecanismos de pesquisa e outros serviços da web.
Os metadados devem ficar dentro da tag head.
precisam ser carregados antes de carregar os dados da pagina

Atributo titulo1
charset Define qual a codificação de caracteres foi usada no documento. Definição de caracteres
content Define o conteúdo associado ao http-equiv. Texto
http-equiv Prove um cabeçalho HTTP com informação do conteúdo do atributo. content-type, default-style, refresh
name Define o nome do metadado. application-name, author, description, generator, keywords, viewport
