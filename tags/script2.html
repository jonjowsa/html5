Sobre a tag Script

Tag usada para definir um script em JavaScript executado no lado do cliente.

A tag permite o script ser escrito dentro dela ou apontar uma fonte externa.

Acesso a todos os atributos globais

Dê preferência para inserir tags scripts no final do fechamento da tag body.

Atributos Exclusivos

Atributos
async Define que o script será executado de forma assincrona. (Funciona apenas para scripts externos) async
charset Define o padrão de caracteres usado. Padrão de caracteres
defer Define que o script só será executado depois da página ser carregada. defer
src Define a fonte externa do script. URL
type Define o tipo de midia do script. Tipo de midia
